# Content Review Checklist: CiviCore Resource and Template Library

Use this checklist before moving any resource draft toward WordPress import, scheduling, or publication.

## 1. Basic Completeness

- [ ] Title is clear.
- [ ] Status is present.
- [ ] Category is present.
- [ ] Slug is present.
- [ ] Meta title is present.
- [ ] Meta description is present.
- [ ] Tags are present.
- [ ] Introduction is useful.
- [ ] Related resources are listed.

## 2. Government Tone

- [ ] Tone is professional.
- [ ] Language is plain and understandable.
- [ ] Content is appropriate for LGU, barangay, water district, small office, or local organization users.
- [ ] Content avoids exaggerated claims.
- [ ] Content avoids aggressive sales language.
- [ ] Content avoids unnecessary technical jargon.

## 3. SEO Readiness

- [ ] Title is search-friendly.
- [ ] Slug is clear and readable.
- [ ] Meta description is useful and specific.
- [ ] Page focuses on one clear topic.
- [ ] Content avoids thin or duplicate material.
- [ ] Headings are practical and easy to scan.
- [ ] Related resources support internal linking.

## 4. Disclaimer Review

- [ ] General disclaimer is included where needed.
- [ ] Stronger disclaimer is included for legal-adjacent templates.
- [ ] Cost, budget, procurement, technical, or compliance disclaimer is included where needed.
- [ ] Content makes no claim of legal sufficiency.
- [ ] Content makes no claim of official approval.
- [ ] Content does not present templates, guides, checklists, or tools as official legal, technical, engineering, accounting, procurement, cybersecurity, data privacy, or compliance advice.

## 5. Publishing Readiness

- [ ] Content is suitable for WordPress draft import.
- [ ] Formatting is clean.
- [ ] Placeholders are understandable.
- [ ] No private or internal project information is exposed.
- [ ] No SheetBot operational details are exposed unless intentionally linked later.
- [ ] Legal-adjacent resources are flagged for extra review before publication.
- [ ] Draft remains in DRAFT status until human review is complete.
